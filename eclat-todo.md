### 🟠 P1：信頼性・購買意欲UP

- **影響**: Persona 04（律子）の共感獲得。
- **実装指示**:

    - 忙しくてスキンケアに時間がない

  - チェックマーク or アイコンで視覚化

#### 5. Solution セクション
- **影響**: Problemからの流れでÉCLATへの期待感を高める。
- **実装指示**:
  - 「ÉCLATが選ばれる理由」の見出し

  - 3つのポイント:
    - 高濃度コラーゲン10,000mg配合
    - 皮膚科医監修・GMP認定
    - 1日1粒、続けやすい

#### 6. Ingredients セクション
- **影響**: Persona 01・03（成分重視層）の購買決定。
- **実装指示**:
  - 背景: ダーク（`--bg-dark`）

    - 低分子化・国産原料 — 吸収率にこだわった処方
  - 各カードにゴールドのアクセントライン

#### 7. Testimonials セクション
- **影響**: Persona 02（あかね）の購買背中押し。

#### 8. How to use セクション
- **影響**: Persona 04（律子）の「続けられそう」という安心感。

## 除外リスト

| 項目 | 除外理由 |
|------|---------|
| 実際の決済機能 | フロントのみでモーダル表示。後から実装可能な設計に |
| 動画素材 | コーチ指定なし（静止画のみ） |
| GA4 / GTM | URL未確定。コメントで挿入箇所を明記 |
| 実際のSNS・LINEリンク | URL未提供。href="#"で仮設置 |
| PWA対応 | LPとしての完成度を優先 |

## 薬機法 NG表現チェックリスト

| NG | OK |
|----|-----|
| シワが消える | 肌にハリを与えるサポート |
| 若返る | エイジングケアをサポート |
| 効果がある | 実感する方が多い |
| 治る・改善する | サポートする・整える |
| 医薬品的効能効果 | 一切使用しない |
